Learn Finnish Game (2022)

This game was a small project to test fun ways of learning a new language. It features a rocket with a frog inside of it, which gains velocity depending on your answer. The player has 5 seconds to choose the correct word, and the languages are swapped after each round. Ultimately, this small project could work as a good primer for learning the specific words in question. The game works by loading two text files which contain the words, and player performance data is saved to a json file, which contains the amount of tries, correct answers, fails and such. A custom class was created for the words which helps keep a track of the data. Although the save data is not even utilized in the actual game, it served as a fun learning opportunity.
