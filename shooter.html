Endless Shooter Prototype (2021)

This endless shooter prototype was created for the purpose of learning and experimenting with different gameplay mechanics. The game has a store from which you can currently buy two upgrades: a rocket launcher and a hook which pulls enemies to you. It also has a 3D text kill feed system, which uses stronger adjectives the higher the kill streak is. You can also walk, sprint, use melee, and aim down the sights. It also features a very high quality player enemy model that was created as a part of a 3D modeling course.

Gameplay

Enemies are randomly spawned around the map, and they keep spawning endlessly based on a timer. Killing an enemy makes them drop orbs, with which you can buy upgrades from the store. These upgrades make the game more fun and allow you to kill enemies faster and with more efficiency.
