Multiplication Game (2022)

A multiplication game built specifically with smartphone screens in mind. It is meant to be a more serious game, oriented towards children to help them learn math. The core gameplay loop works, although the game could use more fun visualizations in order to engage the user more.

Gameplay & Code Example

This function is called after a player presses a button to generate new numbers and increase the played rounds amount. All numbers are first generated and then added to a list, from which the answers are then displayed on the screen. 

public void GenNumbers(int table, int maxRounds, int firstNum, int secNum)
{
    answers.Clear();
    shuffledList.Clear();
    int sum = firstNum * secNum;

    for (int i = 1; i < maxRounds; i++)
    {
        int num = i * table;
        if(num == sum)
            shuffledList.Add(num);
        else
            answers.Add(num);
    }
		
    for (int i = 0; i < 3; i++)
    {
        int num = Random.Range(0, answers.Count);
        shuffledList.Add(answers[num]);
        answers.RemoveAt(num);
    }
}
